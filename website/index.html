The Anti-Framework Framework

You shouldn't need to become a DevOps engineer to deploy your CMS. Horsestrap provides zero-config deployment for .NET 9.0 Umbraco applications. SQLite for local development, SQL Server + Caddy + SSL for production. Build your CMS locally, deploy to Ubuntu in 2 minutes. Simple as that.

What You Get

Umbraco CMS Latest .NET 9.0 with content management
SQL Server Production database with health checks
Automatic SSL Caddy proxy with Let's Encrypt certificates
Zero Downtime Blue/green deployments with rollback
Local Dev SQLite + dotnet run. No Docker needed.

Dead Simple Commands

horsestrap init Create new Umbraco project
horsestrap setup Interactive production deployment
